HTML

Berikut ini adalah pengertian HTML menurut para ahli dalam bukunya.

• Menurut Astamal (2006:8)
Tag dalam hypertext markup language ( HTML) adalah tanda pada dokumen yang memberitahu browser bagaimana merender sebuah halaman web.

• Menurut Lia Kuswayatno
Pengertian hypertext markup language ( HTML) adalah standar untuk membuat halaman web yang memungkinkan pengguna untuk menavigasi dari satu halaman ke halaman lain dengan mengklik link teks.

• Menurut Nugroho (2006c:48)
Pengertian HTML menurut nugroho adalah bahasa pemformatan teks yang digunakan dalam dokumen yang dibagikan melalui jaringan komputer, atau yang biasa dikenal sebagai WWW (World Wide Web).

• Menurut Suyanto (2007:83)
Halaman web merupakan bahasa pemrograman yang digunakan untuk menulis laman website seperti, . HTML, .htm, atau.shtml,.

• Menurut Sidik dan Pohan (2007:9)
Hyper text markup language adalah pengertian dari HTML. Setiap editor teks dapat digunakan untuk menulis file HTML karena mereka hanya teks biasa.

• Menurut Oktavian (2010:13)
HTML merupakan suatu bahasa yang dikenal oleh web browser, agar mendapatkan informasi bahasa yang digunakan untuk membuat konten yang menarik secara visual sebagai lawan dari teks biasa.
